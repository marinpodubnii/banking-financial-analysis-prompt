## üìä Data Quality Checklist

Ensure your data includes:

- [ ] Complete balance sheet with both years
- [ ] Detailed income statement
- [ ] Cash flow statement by activity type
- [ ] Notes on loan portfolio quality
- [ ] Capital adequacy details
- [ ] NPL ratio and provisions
- [ ] Previous year for comparison
- [ ] Currency clearly stated
- [ ] All amounts in same units (thousands/millions)
- [ ] Dates clearly indicated

## üíæ File Formats Accepted

### Preferred:
- **PDF** - Most common for annual reports
- **Excel/XLSX** - For detailed financial data
- **Text/TXT** - For plain data
- **Word/DOCX** - For narrative reports

### Also Supported:
- **CSV** - For tabular data
- **Images (PNG/JPG)** - For scanned documents (OCR will be used)

## üéØ Optimal Upload Strategy

### Option 1: Single Comprehensive Document
Upload one PDF containing:
- Complete annual report
- All financial statements
- All notes and annexes

### Option 2: Multiple Separate Files
Upload separately:
1. `balance-sheet-2023.pdf`
2. `income-statement-2023.pdf`
3. `cash-flow-2023.pdf`
4. `notes-financial-statements.pdf`
5. `annual-report-narrative.pdf`

Copy-paste structured data directly in chat:
```

...
```
